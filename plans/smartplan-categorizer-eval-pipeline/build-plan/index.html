How we get there

Phased build plan

Seven dependency-ordered phases, each independently verifiable, starting with deletion + the cohort query.

Requirements

  1. Work lands in small, independently shippable steps that each leave the build green.
  2. The riskiest integration (live branching) is exercised against a test database before production.
  3. Nothing that runs the agent is ever pointed at real production data during development.

Phases (dependency-ordered)

  1. Blind removal + cohort facade method. Pure deletion plus one new read method (cohort by created-at high-water mark). Independent, unblocks everything. See Blind removal, Cohort.
  2. Eval store schema + version-stamp capture. The two prod tables (incl. r2_fixture_url) and the version-stamp helper (model, prompt, SHA, taxonomy, rules). See Data model.
  3. Eval job (one unattended run). Branch + replay agent + write durable records + dump to SQLite + upload to R2 + delete branch, all in-run; plus fail-closed/no-op handling. See The eval job.
  4. HTML review artifact + email. Extend categorizer_review.py to render legacy-vs-agent; email it to the owner only when the run has disagreements (fully-agreed runs send nothing).
  5. Corrections-derived verdicts + backtest path. Compute corrected vs correct by reading manual recat events after run_at (settling window); add the backtest that hydrates SQLite from r2_fixture_url. See Metrics, Branch lifecycle.
  6. Lazy metrics + versioned trend page. See Metrics.
  7. Move sync + eval to a 12-hour cadence. Update deploy/cron-manager/schedules.json (sync is currently hourly) and add the orphan-branch reaper.

Verification at each step

Per AGENTS.md, before completing any unit of work run from backend/: uv run ruff check ., uv run ruff format --check ., uv run pytest -q. Point any agent-running step at the penny-test Neon branch, never prod (the agent persists). The deletion, schema, and metrics phases are unit-testable offline; the live-run and dump phases need a real branch and R2 bucket and should first be exercised against penny-test.
